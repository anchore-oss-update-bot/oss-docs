Welcome! We appreciate all contributions to Anchore's open source projects.
Whether you're fixing a bug, adding a feature, or improving documentation, your help makes these tools better for everyone.

## Getting Help

The Anchore open source community is here to help. Use [Discourse](https://anchore.com/discourse) for questions, discussions, and troubleshooting.
Use GitHub for reporting bugs, requesting features, and submitting code contributions. See [Issues vs Discussions]({{< ref "issues-and-discussions" >}}) for guidance on which channel to use.

For security vulnerabilities, email [security@anchore.com](mailto:security@anchore.com) - do not create public issues.
See our [Security Policy]({{< ref "security" >}}) for details.
